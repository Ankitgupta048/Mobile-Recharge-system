Mobile Recharge System - Spring Boot (ready)
1) Create MySQL DB: CREATE DATABASE mobile_recharge_system;
2) Update credentials in src/main/resources/application.properties if needed.
3) Build & run: mvn clean package && java -jar target/mobile-recharge-system-1.0.0.jar OR mvn spring-boot:run
4) Open http://localhost:8080
Default admin: admin@example.com / admin123
